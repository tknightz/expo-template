# Clean State Checklist

Run through before ending each session.

## Verification

- [ ] `./verify.sh` passes
- [ ] Dev server starts without errors

## Documentation

- [ ] `agent-progress.md` updated with session record
- [ ] `feature_list.json` reflects actual state
- [ ] No `passing` status without evidence

## Code State

- [ ] No half-finished work left unrecorded
- [ ] No console.log or debug code left in
- [ ] TypeScript has no errors

## Git

- [ ] Changes committed with descriptive message
- [ ] No unintended files staged

## Next Session

- [ ] Next session can run `./init.sh` and start working
- [ ] No manual fixes required to get started
